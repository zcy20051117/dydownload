# 🎵 抖音无水印视频获取工具

本项目用于获取抖音无水印视频。由于抖音存在防爬虫限制，本工具通过模拟浏览器请求的方式获取视频源地址。

## ⚠️ 注意事项
- **视频数量限制**：由于抖音的防爬虫机制，目前只能获取前 **21个** 视频。
- **环境要求**：请确保你的 Chrome 浏览器已经登录了抖音账号。

## 🛠️ 使用步骤

### 1. 获取视频直链
1. 在 Chrome 浏览器中进入 **开发者工具** (F12)，点击 **网络 (Network)** 标签。
2. 在搜索栏中随便搜索一个视频的名称。
3. 点击列表中的第一个请求（名称通常为 `desc`）。
4. 点击 **响应 (Response)**，下滑找到 `video` 里的 `play_addr`。
5. 找到 `url_list`，复制其中的第 1 个网址。
6. 新建一个浏览器页面粘贴进去：
   - 如果什么也没有，请尝试复制第 2 个网址。
   - 打开后，确认是原版无水印视频。
7. **关键步骤**：将这个链接粘贴到代码的 **第 7 行**（即 `GG.get` 的位置）。

### 2. 配置请求头
1. 回到开发者工具，点击 **标头 (Headers)**。
2. 找到 **请求标头 (Request Headers)** 部分。
3. 找到以下三个字段，并将其内容复制粘贴到代码中对应的位置：
   - `cookie`
   - `referer`
   - `user-agent`

### 3. 运行与保存
- 点击运行代码即可。
- **保存路径**：视频会自动保存在项目根目录下的 `video` 文件夹中。
  > **提示**：如果项目文件夹中没有 `video` 文件夹，代码运行时会自动创建，无需手动新建。
